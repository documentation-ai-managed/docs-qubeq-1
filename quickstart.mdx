## Prerequisites

<Callout kind="info" title="Before You Begin">
You need:
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An email address for account verification
- Optional: GitHub, GitLab, or Bitbucket account for repository integration
</Callout>

Qubeq makes it easy to organize your project documentation. Follow these steps to create your account, set up a project space, upload files, and start editing.

## Step 1: Create Your Qubeq Account

Sign up in under a minute.

<Steps>
  <Step title="Visit Qubeq Dashboard" icon="monitor">
    Go to [https://app.qubeq.com](https://app.qubeq.com) and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, create a password, and verify your email address via the confirmation link.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Choose your workspace name and select your role (Owner or Member).
  </Step>
</Steps>

<Callout kind="tip">
Save your login credentials securely. Use a password manager for best security.
</Callout>

## Step 2: Set Up Your First Project Space

Project spaces organize your docs by project.

<Steps>
  <Step title="Create New Space" icon="plus">
    From the dashboard, click **New Space** and enter a name like `My Project Docs`.
  </Step>
  <Step title="Configure Settings" icon="settings">
    Set visibility (Public, Private, or Workspace-only) and add a description.
  </Step>
  <Step title="Invite Collaborators" icon="users">
    Share the space invite link or add users by email.
  </Step>
</Steps>

## Upload Your Initial Documentation Files

Qubeq supports multiple upload methods.

<Tabs>
  <Tab title="Drag & Drop" icon="upload">
    In your project space, drag files directly into the file tree.

    Supported formats: `.md`, `.mdx`, `.txt`, images (`.png`, `.jpg`), and more.
  </Tab>
  <Tab title="Git Integration" icon="github">
    Connect your repo:

````bash
# From space settings, add GitHub repo
git remote add qubeq https://github.com/yourusername/my-docs.git
git push qubeq main
````

    Qubeq syncs changes automatically.
  </Tab>
  <Tab title="CLI Upload" icon="terminal">
    Install Qubeq CLI (coming soon) and run:

````bash
qubeq upload --space "My Project Docs" ./docs/
````
  </Tab>
</Tabs>
