## Overview

Qubeq provides a centralized documentation platform where you can create, organize, and manage all your project documentation. You get a flexible space to build knowledge bases, API references, and user guides with ease. Teams collaborate in real-time, ensuring your docs stay up-to-date and accessible.

Key benefits include intuitive editing tools, version history, search functionality, and seamless integration with your workflows. Whether you're a developer documenting code or a team lead creating onboarding materials, Qubeq streamlines the process.

## Quick Start

Get started with Qubeq in minutes. Follow these steps to create your first documentation page.

<Steps>

<Step title="Sign Up" icon="user-plus">

  Visit `https://app.qubeq.com` and create a free account. Verify your email to unlock full features.

</Step>

<Step title="Create Workspace" icon="plus">

  Set up a new workspace for your project. Choose a name like `MyProjectDocs`.

</Step>

<Step title="Add First Page" icon="file-plus">

  Click "New Page" and start editing with markdown. Here's a simple example:

  <CodeGroup tabs="Markdown,Preview">

  ````markdown
  # Welcome to My Docs

  This is your first page.

  ## Quick Links

  - [Installation](/installation)
  - [API Reference](/api)
  ````

  ````

  Preview shows:

  # Welcome to My Docs

  This is your first page.

  ## Quick Links

  - [Installation](/installation)
  - [API Reference](/api)

  ````

  </CodeGroup>

</Step>

<Step title="Publish and Share" icon="globe">

  Hit "Publish" to make your docs live. Share the URL: `https://docs.qubeq.com/your-workspace`.

</Step>

</Steps>
