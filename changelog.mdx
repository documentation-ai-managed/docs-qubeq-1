## Upgrade to the Latest Version

Follow these steps to upgrade your Qubeq instance and benefit from the latest improvements.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup of your documentation space.

    ```
    # Example backup command
    qubeq backup --output ./backups/latest.tar.gz
    ```

  </Step>
  <Step title="Update Package" icon="package">

    <Tabs>
      <Tab title="npm" icon="nodejs">
````bash
npm install qubeq@latest
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn add qubeq@latest
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Apply any database schema changes.

    ```
    qubeq migrate --env production
    ```

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

    Restart your Qubeq server.

    ```
    systemctl restart qubeq
    # Or docker restart qubeq-container
    ```

  </Step>
</Steps>

<Callout kind="tip">
After upgrading, verify your custom configurations in the dashboard at `https://dashboard.example.com/settings`.
</Callout>
